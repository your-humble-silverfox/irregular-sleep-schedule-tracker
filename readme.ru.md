# Трекер Иррегулярного Цикла Сна
Language: [English](readme.md) | [Русский](readme.ru.md)

`Трекер нерегулярного цикла сна` позволяет вести учет циклов сна и бодрствования, а также сравнивать их с целевыми значениями.

### Статус разработки

#### Реализовано:

На данный момент (03.07.2026) реализован следующий функционал:

- Первичное конфигурирование программы
- Хранение данных сна/бодрствования
- Сравнение фактических данных с целевыми
- Прогнозирование периодов сна/бодрствования

#### Требует реализации:

- Хранение и визуализация прогнозов
- Работа приложения в фоновом режиме/трее
- Автоматическая отметка о начале периода сна на случай ошибки пользователя

### Зависимости

- `PyQT6`
